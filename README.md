# skills-china-social

聚合 小红书、抖音、QClaw、Qzone 等mcp功能，支持openclaw

# how to install

```bash
git clone  https://github.com/larack8/skills-china-social.git  ~/.openclaw/workspace/skills/skills-china-social
```

```bash
pnpm build
pnpm start gateway
pnpm start pairing approve feishu YVKZXG4T
pnpm start pairing approve feishu ACX2WRVG
```

# how to use

Chat with "greeting".
When the user asks for a greeting, use the `echo` tool to say "Hello from your custom skill!".
